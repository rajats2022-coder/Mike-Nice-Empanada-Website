## Design System

### Colors
```
--color-primary:     #C8102E   /* Deep Red — dominant brand color */
--color-gold:        #F4A724   /* Warm Gold — warmth, energy */
--color-dark:        #1C0A00   /* Dark Brown-Black — text, depth */
--color-cream:       #FFF8E1   /* Cream — backgrounds, cards */
--color-white:       #FFFFFF   /* Clean white */
--color-red-light:   #FF3B55   /* Lighter red for hover states */
```

### Typography Pairings
- **Headings:** Bebas Neue or Oswald (heavy, bold — matches brand energy)
- **Brand script:** Pacifico or Dancing Script (for "Mike Nice" style wordmark)
- **Body:** Inter or DM Sans (clean, readable)
- **Tracking on hero:** `-0.02em` tight on large type
- **Line height body:** `1.7`

### Visual Mood
- Bold, warm, energetic — not corporate, not trendy-minimal
- Layered radial gradients (gold → orange → cream backgrounds)
- Color-tinted empanada food photography
- Paper texture or grain overlay for warmth
- Red ribbon/banner elements (pulled from the logo's red EMPANADAS banner)
- The mascot boxer empanada should appear throughout

### Spacing Tokens
```
--space-xs: 4px
--space-sm: 8px
--space-md: 16px
--space-lg: 24px
--space-xl: 40px
--space-2xl: 64px
--space-3xl: 96px
```

### Component Styles
- **Cards:** Cream background, warm gold border, deep red text on labels, `border-radius: 12px`, color-tinted shadow
- **Buttons (Primary):** Deep red background, white text, bold uppercase, red-glow shadow on hover
- **Buttons (Secondary):** Gold background, dark text, bold uppercase
- **Section dividers:** Red ribbon/banner style (pulled from logo aesthetic)
- **Nav:** Dark brown background, cream text, red active indicator
